# Communication Helper

A simple touch-friendly app for people with limited verbal communication.

## Features

- Large icon buttons for: Yes, No, Pain, Nurse, Choking
- Buttons speak the selected word aloud
- Easy finger tapping with high-contrast labels
- Works in modern browsers with Speech Synthesis support

## Run locally

1. Open `index.html` in a browser.
2. Tap any icon to hear the word spoken.

## Install as an app

- In supported browsers, use the browser menu to "Install" or "Add to Home screen".
- On Apple devices, open the page in Safari and choose "Add to Home Screen".

## Sentence Builder

- Open `sentences.html` to build a sentence using common words.
- Tap words to add them to the sentence, then use "Speak Sentence".

## Files

- `index.html` — app structure
- `styles.css` — responsive touch-friendly styling
- `script.js` — speech and interaction logic
